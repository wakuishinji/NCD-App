NCD 管理ハブ

Nakano Clinic Database

管理業務をカテゴリ別に整理しました。必要なカードをクリックして各マスター/設定画面へ移動してください。

診療所・標榜診療科

診療所リスト

登録済み診療所の検索・エクスポート・削除にアクセスします。

標榜診療科分類マスター

標榜診療科で利用する分類ラベルを追加・改名・削除します。

診療形態マスター

オンライン診療・夜間診療などの診療形態タグを管理し、公開可否や表示順を調整します。

説明候補の管理

診療・検査ごとの説明候補は、それぞれのマスター管理画面から直接追加・承認できます。

旧「説明テンプレート管理」画面は廃止しました。

個人資格・医療分野

個人資格マスター

分類・医療分野・名称・学会名を編集し、承認ステータスを管理します。

学会マスター

分類ごとの学会名を整理し、個人資格入力時の候補を管理します。

医療分野マスター

個人資格で利用する医療分野カテゴリを管理します。

施設認定・種類マスター

施設認定マスター

種類・名称・備考を管理し、候補から承認までの状態を操作します。

施設認定種類マスター

施設認定で使用する種類カテゴリを登録・整理します。

検査マスター・検査分類

検査マスター

検査名称の一覧管理・検索・CSV入出力を行います。

検査分類マスター

検査マスターで使用する分類ラベルを編集します。

症状・部位マスター

症状マスター

症状の基礎情報や対応部位・推奨診療／検査を編集します。

部位マスター

解剖部位の階層・患者向け名称・別名を管理し、症状との紐付けに備えます。

アカウント・権限

管理者申請の承認

医師会スタッフからの管理者権限リクエストを確認し、施設管理者ロールを付与・却下します。

診療マスター・診療分類

診療マスター

診療メニューの分類・名称・説明を管理します。

診療分類マスター

診療マスターで使用する分類ラベルを管理します。

予防接種・健診マスター

予防接種マスター

ワクチン種類別に名称・詳細を整理します。

予防接種種類マスター

予防接種で利用する種類ラベルを管理します。

健診マスター

健診種類別の名称・詳細を整理します。

健診種類マスター

健診に使用する種類ラベルを管理します。

運用補助

運用ToDo管理

運用タスクを一覧し、Cloudflare Workers に保存します。

API / 環境設定

AIモデルやプロンプトなどの共通設定を編集します。

厚労省ID同期（systemRoot専用）

厚労省の施設IDを既存診療所へ紐付け、公開データから住所・電話などを反映します。

※このカードは systemRoot のみ表示されます。
